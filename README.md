# -Criando-Um-Organizador-de-Declara-o-de-Imposto-de-Renda
Com base nos conteúdos apresentados nas aulas ministradas pelo professor Felipe Aguiar, este projeto consiste no desenvolvimento de uma planilha em Excel voltada para a organização e consolidação das informações necessárias à Declaração de Imposto de Renda.  A solução foi construída aplicando, de forma prática.
