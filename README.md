# Criando Um Organizador de Declaração de Imposto de Renda
Com base nos conteúdos apresentados nas aulas ministradas pelo professor Felipe Aguiar, este projeto consiste no desenvolvimento de uma planilha em Excel voltada para a organização e consolidação das informações necessárias à Declaração de Imposto de Renda.

A solução foi construída aplicando, de forma prática, os conceitos abordados ao longo do curso, com foco na criação de um agregador de dados funcional, validado e de fácil utilização. A planilha permite o controle eficiente de rendimentos, despesas e demais informações relevantes, utilizando recursos como validação de dados, menus de navegação, formatações automáticas e links rápidos.

O projeto reforça o uso do Excel como uma ferramenta robusta para organização financeira e demonstra a capacidade de estruturar soluções práticas, intuitivas e alinhadas às boas práticas apresentadas em aula.
